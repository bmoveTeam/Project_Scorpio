INFORMACION SOBRE EL RETO

Reto 8:roboAlmi

En este reto necesitamos 4 ordenadores. En un ordenador deberemos instalar Untangle, un sistema operativo para hacer un portal cautivo y tendrá 2 tarjetas de red. En otro ordenador instalaremos Owncloud y crearemos un DNS con el nombre del grupo. También necesitaremos dos Robots con arduino para que uno sea un siguelineas y otro para controlarlo con el móvil. Crearemos una Wifi segura para que no se pueda hackear el robot. Haremos un video presentación de los robots y una web presentación del reto. Justo debajo esta es la arquitectura de red que hemos usado utilizando Packet Tracer
